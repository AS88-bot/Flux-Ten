## 🔹 Description
A cinematic, interactive countdown where numbers emerge from a living particle system. 
Each transition dissolves into motion—dispersing, swirling, and reforming through smooth, physics-inspired animation.

Designed to feel calm, fluid, and immersive, the experience blends interaction, depth, and generative motion into a single evolving visual.

## 🔹 Features
- **Particle-driven rendering** — 3,000+ particles dynamically form each number
- **Interactive motion** — Cursor/touch introduces real-time repulsion and flow
- **Cinematic transitions** — Swirl-based morphing between numbers
- **Smooth countdown** — Continuous progression from 10 → 1
- **Depth & layering** — Multi-plane particle system for spatial richness
- **Subtle sound design** — Lightweight, synchronized audio feedback

## 🔹 Tech Stack
- **Core**: TypeScript, React
- **Rendering**: HTML5 Canvas
- **Styling**: Tailwind CSS
- **Audio**: Web Audio API

## 🔹 Inspiration
Created for the **Google I/O 2026 Countdown Challenge**, exploring motion, interaction, and generative design in a real-time visual system.
